API - The API is work in progress and is subject to change during development.
  • /stats global pool stats
  • /stats global block stats
  • /pool_stats - historical stats
  • /payments - payment history
  • /worker_stats?taddr - historical time per pool json
  • /live_stats - live stats
  • /api/workers_final - workers stats for GW_Dashboard
  • /api/keepalive_worker?taddr - keepalive shares 10minutes without shares
  • /api/payouts_worker?taddr - payouts by address
